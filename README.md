# منصة إبداع - Ebdaa Platform

منصة عربية ديناميكية لنشر الإبداع والكتابة والتصميم. تجمع بين الكتاب والمصممين والقراء في مجتمع واحد.

## 🎯 الميزات الرئيسية

- ✅ نظام تسجيل دخول وإنشاء حساب آمن
- ✅ إدارة الكتب والتصاميم
- ✅ نظام المبدعين والملفات الشخصية
- ✅ البحث والتصفية المتقدمة
- ✅ واجهة مستخدم عصرية وسهلة الاستخدام
- ✅ دعم كامل للغة العربية (RTL)
- ✅ أمان عالي ضد الهجمات

## 📋 المتطلبات

- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- Apache مع mod_rewrite مفعل
- Composer (اختياري)

## 🚀 التثبيت

### 1. استنساخ المشروع

```bash
git clone https://github.com/yourusername/ebdaa-platform.git
cd ebdaa-platform
```

### 2. إنشاء قاعدة البيانات

قم بتشغيل ملف إنشاء قاعدة البيانات:

```bash
php config/init_database.php
```

### 3. إضافة البيانات التجريبية (اختياري)

```bash
php config/sample_data.php
```

### 4. تكوين الملفات

تأكد من أن الملفات التالية لها صلاحيات الكتابة:

```bash
chmod -R 755 uploads/
chmod -R 755 config/
```

### 5. تشغيل الخادم

```bash
php -S localhost:8000
```

ثم افتح المتصفح على: `http://localhost:8000`

## 🔐 الأمان

### ميزات الأمان المطبقة:

1. **حماية من SQL Injection**: استخدام Prepared Statements
2. **حماية من XSS**: تنظيف المدخلات وتشفير المخرجات
3. **حماية من CSRF**: استخدام tokens
4. **تشفير كلمات المرور**: استخدام bcrypt
5. **حماية المجلدات**: ملف .htaccess
6. **رؤوس الأمان**: X-Frame-Options, X-Content-Type-Options
